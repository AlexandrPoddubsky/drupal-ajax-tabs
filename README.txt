Description
===========
The module ajaxifies MENU_LOCAL_TABS links.


Dependencies
============
Ajax Load (http://drupal.org/project/ajax_load)


Installation
============
- Drop the module into sites/all/contrib/modules or wherever else.
- Enable the module from admin/build/modules or 'drush en ajax_tabs -y'.
- Goto admin/settings/ajax-tabs and configure its settings.
